<p>The completed action of observing, monitoring, measuring or otherwise qualitatively or quantitatively gathering data or information about one or more aspects of a subject's physiologic or psychologic state.<br/></p><p><br/></p><p>For example, lab test, taking vital signs, physical exam, etc.<br/></p><p><br/></p><p>NOTE:  The ActivityRelationship allows the model to handle batteries and panels that are composed of component LabTests (for example).  Flow cytometry and gene expression analyses are examples that would make use of this association.<br/></p>

BRIDG THC anticipates renaming this attribute to be more specific for a PerformedObservation.<br /><br />NOTE: Because this is of datatype CD, it contains both a coded and textual representation of the name of the PerformedObservation.
public ST
  nameModifiedText
Notes: A character string that is a revision of the original text observation to enable the coding of the text. <br /><br />For example, if the originalText is "hedache", the nameModifiedText could be changed to "headache", so that the text can be successfully coded.<br /><br />NOTE: In the context of BRIDG, text modification occurs a single time for a given instance of OriginalText.
public DSET
  methodCode
Notes: The means or technique used to ascertain the observation.<br /><br />For example, blood pressure measurement method could be arterial puncture or sphygmomanometry.<br /><br />For example, global introspection, algorithm, bayesian to assess AE causality.<br /><br />For example for a clinical result assay method, values could include: Estrogen Receptor Assay, Progesterone Receptor Assay, p53 Assay, etc.
