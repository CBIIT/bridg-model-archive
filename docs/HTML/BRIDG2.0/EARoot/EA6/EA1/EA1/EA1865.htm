<p>The collection of documents provided to the regulatory authority at one time.<br/></p><p><br/></p><p>NOTE: A submission unit is made up of one to many documents. Properly defined, the submission unit concept enables companies to create new submission units from any combination of new and previously submitted document.<br/></p><p><br/></p><p>NOTE: In the dynamic aspects of the model, a submission unit is one message that may have a collection of many documents. There are rules for how submission units are evaluated are described in a state diagram, and the receipt date of the submission unit "starts the clock" for the review of the contents of the submission unit. <br/></p>
