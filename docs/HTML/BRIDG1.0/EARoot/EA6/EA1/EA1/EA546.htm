<p>Used to describe a period of time that cuts across the arms of a study. Assuming time is depicted horizontally, an Epoch is a vertical segmentation of one or more arms which are parallel in time. Epochs typically group similar activities (i.e. screening, treatment, follow-up). Participants in a study move through epochs serially over time, and no participant can be assigned to more than one epoch at any particular time.<br/></p><p><br/></p><p>For example, an Epoch could be called "Screening", "Treatment", or "Follow Up".<br/></p><p><br/></p>
