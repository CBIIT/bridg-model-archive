<p>The collection of documentation's provided to the Regulatory Authority at one time.<br/></p><p><br/></p><p>A submission unit is made up of one to many documentation's. Properly defined, the submission unit concept enables companies to create new submission units from any combination of new and previously submitted documentation.<br/></p><p><br/></p><p>In the dynamic aspects of the model, a submission unit is one message that may have a collection of many documentation's. There are rules for how submission units are evaluated are described in a state diagram, and the receipt date of the submission unit "starts the clock" for the review of the contents of the submission unit. <br/></p>
