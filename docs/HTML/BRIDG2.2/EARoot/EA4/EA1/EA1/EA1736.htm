<p>The completed activity which marks a StudySubject-specific, protocol-defined point in the course of a study.<br/></p><p><br/></p><p>For example, obtain informed consent, verify eligibility criteria, enroll, registration to a study, randomize, assignment to a treatment arm, start of on-study period, complete study visits, end of on-study period, exit trial, break treatment blind, protocol violation, premature withdrawal, etc.<br/></p>
