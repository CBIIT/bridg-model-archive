<p>Relates an activity to a particular encounter with a StudySubject or ExperimentalUnit.  The design of an encounter can be visualized as cells in a matrix whose columns are time points and whose rows are activities; each cell of such a matrix is an SOA cell; and an X in a particular cell indicates that an activity (row) is to take place at point in time (column).<br/></p><p><br/></p><p>For example, a CBC is to occur during the week 2 encounter. <br/></p>
