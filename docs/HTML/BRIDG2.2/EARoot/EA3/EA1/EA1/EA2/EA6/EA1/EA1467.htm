Appears In: caBIG - Load Labs In CDMS
<p>STORYBOARD:  CRA looks at the schedule of a given patient on a study for a specific visit.  She identifies all the lab tests that should have been scheduled/ completed for that visit.  CRA would need all the lab values in a given timeframe that would fall within the timeframe of the visit.  CRA then reviews the lab tests/values and identifies which ones are relevant to be loaded into the CTMS (for example, a diabetic pt may have also had a A1C – in this case, this values would not be stored in the CTMS)<br/></p>
