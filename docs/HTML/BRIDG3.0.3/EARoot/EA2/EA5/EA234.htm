Constraint Type Status Detail
Is Participated In By Qualifier Invariant Approved
Associations from Subject, StudySubject and ExperimentalUnit are valid (only BiologicEntity, not Product).
relativeToReference Qualifier Invariant Approved
endRelativeToReference must not logically precede the startRelativeToReference. For example, if startRelativeToReference is "during", endRelativeToReference must not be "before".
Dose Expression Qualifier Invariant Approved
If a dose expression uses parameters, each parameter must be defined as a DefinedExpressionVariableRelationship.localVariableName.
Active Ingredient Qualifier Invariant Approved
The attributes: activeIngredientDose, periodActiveIngredientDoseTotal must only be used when the PerformedSubstanceAdministration is associated to a StudyAgent or ConcomitantAgent that has only one active ingredient. Otherwise, it would be ambiguous to which active ingredient the amounts indicated in these attributes apply.
Dose Qualifier Invariant Approved
The attributes: productDose, periodProductDoseTotal, treatmentVehicleAmount, activeIngredientDose, periodActiveIngredientDoseTotal must only be used when the PerformedSubstanceAdministration is associated to a single StudyAgent or ConcomitantAgent. Otherwise, it would be amiguous to which agent the amounts indicated in these attributes apply.
