<p>This view represents the Adverse Event (AE) semantics from the Adverse Event Domain Analysis Model, which was developed by representatives from several AE projects: HL7's Individual Case Safety Report (ICSR), CDISC's Study Data Tabulation Model for Human Subjects (SDTM), NCI's Cancer AE Reporting System (caAERS) and Adverse Event Expedited Reporting System (AdEERS), and the Federal Adverse Event Task Force (FAET).<br/></p><p><br/></p><p>The AE model introduced several new concept areas pertaining to devices, animals, and products.  <br/></p><p><br/></p><p>There are 2 instance diagrams that illustrate the use of this view.<br/></p>
